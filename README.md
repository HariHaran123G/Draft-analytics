# ⚽ Football Sports Analytics using Python

A data analytics project focused on football match performance analysis using Python, Pandas, NumPy, and Matplotlib.  
This project explores team performance, possession efficiency, aggression metrics, attendance impact, and goal-scoring trends through data visualization and statistical analysis.

---

## 📌 Project Objectives

- Analyze top goal-scoring football teams
- Study possession vs goal-scoring relationships
- Identify aggressive playing styles using foul and card metrics
- Measure crowd attendance impact on team performance
- Evaluate possession efficiency across teams

---

## 🛠 Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Excel Dataset

---

## 📊 Key Analysis Performed

### 1️⃣ Top Goal Scoring Teams
- Combined home and away goals
- Ranked teams based on total goals scored
- Visualized Top 10 scoring teams using bar charts

### 2️⃣ Possession vs Goals Analysis
- Analyzed relationship between ball possession and goals scored
- Compared average possession between winners and losers
- Used scatter plots and grouped statistical analysis

### 3️⃣ Aggressive Team Analysis
- Calculated aggression score using:
  - Fouls
  - Yellow cards
  - Red cards
- Identified teams with most aggressive playing styles

### 4️⃣ Attendance vs Performance Analysis
- Created custom performance score using:
  - Goals
  - Possession
  - Shots
  - Passing
  - Red card
